**Gilt Counselling Management System – Summary**

The system supports **user registration, appointment booking, and role-based dashboards** for users, admins, and counselors.

* **User Registration**

  * *Online self-registration*: Users sign up via email or Google, verify OTP, receive a welcome email, complete their profile, and access the user dashboard to book appointments.
  * *Office registration*: Admin creates accounts for walk-in users, sends access instructions via email, and can immediately book appointments on their behalf.

* **Login & Role Management**

  * Users are redirected to the appropriate dashboard based on role: User, Admin, or Counselor.

* **Appointment Booking**

  * Users can book appointments online by selecting counselling type, available date and time, adding optional notes, and confirming. The system validates availability, creates a pending appointment, and sends notifications (email, push, admin alert).
  * Users can view all appointments and their statuses.

* **Appointment Management**

  * Users can view upcoming, past, and cancelled appointments, with options to reschedule or cancel.

* **Admin Dashboard**

  * Provides statistics (appointments, pending requests, total users), today’s appointments with quick actions, recent users, notifications, and shortcuts to key tasks (add user, book appointment, add counselor, send notifications).

* **Admin User Management**

  * Admins can view all users and edit user profiles, including personal details, contact information, and profile photos.
